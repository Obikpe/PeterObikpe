Project: COVID-19 Data Analysis

Project Overview

This project involves analyzing COVID-19 data using SQL. The analysis explores the spread of the virus, mortality rates, and vaccination efforts. The findings can be valuable for policymakers, healthcare organizations, and the public in understanding the dynamics of the pandemic and making informed decisions.

Project Details

Key Objectives:

  • Selecting relevant data from the `CovidDeaths` and `CovidVaccinations` tables.
  • Calculating key metrics such as death percentage and infection percentage.
  • Filtering data to focus on specific locations (e.g., Nigeria).
  • Identifying countries with the highest infection and death rates.
  • Analyzing COVID-19 data by continent.
  • Examining global COVID-19 numbers on a daily and overall basis.
  • Comparing total population with vaccination data, including using CTEs and temporary tables.
  • Creating views to store data for future visualization.

Data Sources:

The analysis uses data from two tables:

  • `CovidDeaths`: Contains information about COVID-19 cases and deaths.
  • `CovidVaccinations`: Contains data related to COVID-19 vaccinations.

You can find the SQL code used for this analysis here.

Key SQL Queries and Analysis:

  • Data Selection and Ordering: Selecting specific columns (`Location`, `Date`, `total_cases`, `new_cases`, `total_deaths`, `population`) from the `CovidDeaths` table and ordering the results by location and date.
  • Total Cases vs. Total Deaths: Calculating the death percentage for each location and date to understand the mortality rate of COVID-19. The query filters for Nigeria.
  • Population that has COVID: Calculating the percentage of the population that has been confirmed to have contracted COVID-19.
  • Highest Infection Rate: Identifying the location with the highest infection rate compared to its population. The query filters for Nigeria.
  • Highest Death Rate: Determining the location with the highest death rate.
  • Analysis by Continent: Analyzing total deaths and death counts by continent.
  • Global Numbers: Calculating daily and overall global COVID-19 cases, deaths, and death percentages.
  • Total Population vs. Vaccinations: Comparing population data with vaccination data. This includes calculating the rolling number of people vaccinated using window functions, CTEs, and temporary tables.
  • Creating Views: Creating a view (`PercentagePopulationVaccinated`) to store the results of the population vs. vaccination analysis for future use, such as in data visualization.

Key Insights

  • The analysis provides insights into the spread of COVID-19, showing how the virus has affected different populations over time.
  • Mortality rates vary by location, highlighting differences in healthcare systems and public health responses.
  • Vaccination efforts are quantified, showing the percentage of the population vaccinated, which is crucial for understanding progress in controlling the pandemic.
  • Global trends, including daily and total cases and deaths, provide a broader context for understanding the pandemic's impact.

Conclusion

This project demonstrates the use of SQL to analyze COVID-19 data, providing valuable insights into the pandemic's progression and the effectiveness of vaccination campaigns. The results of this analysis can be used to inform public health policy and improve responses to similar crises in the future.
